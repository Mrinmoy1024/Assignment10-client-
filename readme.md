#  HabitTracker

A modern, full-stack habit tracking web application built with React, Firebase, and a Node.js/Express backend powered by MongoDB — helping you build consistency, one day at a time.

---

## Features

- **Daily Habit Tracking** — Log and monitor your habits every day
- **Firebase Auth** — Secure user authentication and persistent data
- **Progress Visualization** — Visual streaks and completion stats
- **Polished UI** — Smooth animations powered by Framer Motion
- **Fully Responsive** — Works great on mobile and desktop
- **Toast Notifications** — Real-time feedback with React Toastify
- **DaisyUI + Tailwind CSS v4** — Beautiful, themeable component system

---

## Tech Stack

| Category | Technology |
|---|---|
| Framework | React + Vite |
| Styling | Tailwind CSS + DaisyUI |
| Auth | Firebase |
| Backend | Node.js + Express.js |
| Database | MongoDB |
| Middleware | CORS |
| Routing | React Router 7 |
| Animation | Framer Motion |
| UI / UX | React Icons, React Tooltip, SweetAlert2, Swiper |
| Notifications | React Toastify |
